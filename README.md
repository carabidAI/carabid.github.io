# Carabid

Minimal [carabid.ai](https://carabid.ai) landing page (stealth).

- **Stack:** Jekyll, HTML/CSS/JS
- **Hosting:** GitHub Pages

## Local development

```bash
bundle install
bundle exec jekyll serve
```

Open http://localhost:4000

## Build

```bash
bundle exec jekyll build
```

Output is in `_site/`.
